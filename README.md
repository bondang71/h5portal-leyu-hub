# h5portal-leyu-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源形式存储，可作为独立入口或功能模块使用。

## 项目简介

`h5portal-leyu-hub` 是一个轻量级的 HTML 页面集合仓库。它不绑定特定域名或网站，旨在提供一个集中存放、版本管理和快速访问静态页面的空间。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开或通过托管服务部署。

## 目录说明

```
h5portal-leyu-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个页面以独立文件形式存在，文件名即为页面标识。
- `assets/`：存放多个页面可能共用的静态资源，避免重复。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容可能包含交互功能，但功能范围仅限于前端实现。
- 归档的页面可能来自不同时期或不同场景，彼此之间无直接关联。

## 维护说明

- 本仓库主要面向开发者和维护者，用于页面版本追踪和内容更新。
- 如需添加新页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保其自包含性。
- 公共资源文件更新时，请注意检查各页面引用是否同步。
- 欢迎提交 Issue 或 Pull Request 协助完善内容。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/h5portal-leyu-hub.git
   ```
2. 直接打开 `pages/` 下的任意 HTML 文件即可查看。
3. 如需部署，可将整个仓库上传至静态文件托管服务。
